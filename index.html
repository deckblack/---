Строительные HTML теги:

Header - используется на странице один раз, для верхней части сайта, шапки.

Footer - используется на странице один раз, для нижней части сайта, подвала.

Nav - используется на странице для меню навигации сайта.

Div - элемент является блочным элементом и предназначен для выделения фрагмента документа с целью изменения вида содержимого. Как правило, вид блока управляется с помощью стилей. Чтобы не описывать каждый раз стиль внутри тега, можно выделить стиль во внешнюю таблицу стилей, а для тега добавить атрибут Class или Id с именем селектора.

HTML теги для работы с контентом:

H1, H2, H3, H4, H5, H6 - теги заголовков, тег H1 используется один раз на странице, остальные теги можно использовать многократно, не рекомендуется использовать более 3-х разных заголовков на странице.

P - тег абзаца, в него помещается обычный текст.

Span - строчный тег для выделения текста и применения к нему каких либо атрибутов.

A - тег создает ссылку которая помещается в атрибут href. Внутри элемента может быть как текст так и картинка. target="_blank" открывает ссылку в новой вкладке. Атрибут download предлагает скачать то что находится по ссылке.

Img элемент отвечающий за вставку картинки. Атрибут src="" это ссылка на картинку, alt="" - описание картинки.

Ul - маркированный список. Внутри располагается тег Li, в котором перечисляется содержимое списка.

    Формы ввода:

    Все теги для ввода данных помещаются внутри элемента Form который имеет обязательный атрибут Action="" в котором указывается ссылка на адрес сервера, где будут храниться или обрабатываться введенные данные. Атрибут Method указывает каким образом будут передаваться данные, открытым - get или скрытным - post. Только скрытным путем можно отправлять большие объемы данных и прикрепленные файлы. При это обязательно использовать enctype="multipart/form-data". Все элементы внутри form можно блокировать атрибутом disabled делая их серыми и неактивными.

    Input - поле для ввода каких либо данных пользователем. Содержит в себе атрибуты Name - название для элемента, Value - значение элемента. Атрибут Placeholder задает для формы ввода отображаемое название подсказку.
    Тип данных опредляется отдельным атрибутом type="", который может содержать в себе:

    • Text - если пользователь будет вводить текст.


    • Tel - если пользователь будет вводить цифры на телефоне, это переключит его раскладку автоматически на цифровую.

    • Password - если пользователь будет вводить пароль, это замажет вводимые в строке символы.


    • Radio - используя это значение атрибута, вы должны поместить весь элемент Input в тектовый тег P, внутри которого будете задавать названия для кажого отдельного radio. Тем самым вы создадите подобие списка, который будет объединятся в группу одинаково названным атрибутом Name, но атрибут Value должен отличаться и соответствовать названию указанному в атрибуте P. Этот элемент дает пользователю выбирать только один пункт из списка. Атрибут Checked заданный у одного из Radio делает этот пункт выбранным по умолчанию.

      Мужчина

      Женщина

      Робот

    • Checkbox - тоже помещается в тег P. Но в отличии от radio объединяется одинаковым атрибутом Value, а атрибут Name для каждого пункта должен быть разным. Этот элемент позволяет выбирать сразу несколько пунктов. Атрибут Checked тут тоже работает.

      Вы сильный?

      Вы быстрый?

      Вы человек?

    • File - позволяет прикреплять файлы и отправлять их на сервер. Атрибут Value тут не используется.

    Textarea - позволяет вводить многострочный текст. Атрибуты Cols, Rows можно не использовать если вы будете задавать размеры используя CSS. Атрибут Placeholder задает для формы ввода отображаемое название подсказку.

    Select - выпадающий список, должен содержать в себе тег Option, внутри которого перечисляются все пункты выпадающего списка. У select может быть атрибут multiple позволяющий выбирать из списка сразу несколько пунктов с зажатым ctrl. У option может быть атрибут selected который задает какой либо пункт списка по умолчанию.

    Button - создает кнопку. Тип кнопки задается атрибутом Type и может быть:

    • Submit - отправляет всю форму.
    • Reset - сбрасывает все значения формы по умолчанию.

    CSS - каскадные таблицы стилей:

    CSS записываются в отдельном файле с расширением .css и зыписываются в виде:

    селектор {
    свойство: значение;
    свойство + значение = параметр;
    }

    Селектор может записываться несколькими способами:

    • тег - обращение ко всем тегам
    • .класс - обращение ко всем классам
    • тег.класс - обращение ко всем классам внутри тега
    • .класс тег - обращение ко всем тегам внутри класса
    • .класс>тег - обращение ко всем тегам на первом уровне класса
    • * - обращение вообще ко всем тегам независимо от названия
    • тег, тег, .класс, .класс - обращение сразу к нескольким тегам или классам

    Подключаются CSS с помощью тега Link который располагается в Head и имеет атрибут rel="stylesheet" обозначающий что это CSS, а так же атрибут href="" указывающий ссылку на CSS файл.

    CSS параметры для работы со шрифтом и текстом:

    font-family: имяшрифта, "имя шрифта", тип шрифта; - задает тексту шрифт, названия шрифтов перечисляются через запятую, если в названии есть проблел название заключается в кавычки. В конце указывается тип шрифта: serif, sans-serif, monospace, cursive, fantasy.
    Атрибут inherit указывает наследуется ли шрифт от родительского элемента.

    font-size: размер; - указывается размер шрифта в различных единицах.

    font-style: стиль; - указывает стиль шрифта:

    • italic - курсивный
    • oblique - наклонный
    • normal - обычный

    font-weight: толщина; - указывает на толщину шрифта:

    • 100 - thin
    • 300 - lite
    • 400 - normal
    • 500 - medium
    • 600 - semibold
    • 700 - bold
    • 900 - black

    color: цвет; - указывает на цвет шрифта.

    • #000000 - шестнадцатеричный код
    • rgb (0, 0, 0) - rgb
    • black - просто указание цвета

    text-align: выравнивание; - горизонтальное выравнивание текста:

    • center - по центру
    • justify - по всей ширине
    • left - слева
    • right - справа

    text-decoration: значение1 значение2; - декорирование текста, можно применять сразу несколько зачений:

    • line-through - перечеркнутый текст
    • overline - линия проходит над текстом
    • underline - линия подчеркивает текст
    • none - убирает декорирование, в том числе у ссылок

    text-shadow: x, y, размытие, цвет; - создает тень у текста, например text-shadow: 1px, 1px, 10px, #000000;

    text-transform: значение; - указывает прописные или строчные символы:

    • capitalize - первый символ кажодого слова будет заглавным
    • lowercase - все символы строчные
    • uppercase - все символы прописные
    • none - обычное поведение символов

    text-indent: отступ; - размер красной строки у текста.

    letter-spacing: значение; - устанавливает отступы между символами.

    word-spacing: значение; - устанавливает расстояние между словами.

    white-space: nowrap; - запрещает перенос строки, значение normal разрешает перенос строки.

    line-height: значение; - устанавливает межстрочный интервал текста.
